# foxyon
Meu projeto para apresentar aos entrevistadores

# Um anti-DoS performático baseado em HashCash
Foxyon é extremamente performático, tanto em questão de memória quanto de processamento. Em testes, aguentou e superou +100.000 requisições por segundo utilizando 28% de 3 vCPU de uma máquina virtual com 4 GB de RAM, utilizando somente 14 MB durante sua execução.

É feito para acoplar ao "auth_request" do Nginx, não sendo extremamente modular e cumprindo a versão de gerenciamento de sessão eficiente.
